Aide

comment utiliser le site web

  • Démarrer : Cliquez sur la boîte centrale pour lancer l'expérience après la vidéo d'introduction.
  • Explorer : Cliquez sur les cartes rotatives pour découvrir des informations détaillées.
  • Navigation : Utilisez la barre en haut ou le menu burger sur mobile.
  • Fermer : Cliquez sur "✖" ou en dehors des overlays pour les fermer.
  • Interagir : Passez la souris pour voir des animations.
